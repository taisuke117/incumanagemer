# incumanagemer — 細胞パッセージ管理タイムライン

細胞培養の継代（パッセージ）を、播種 → 回収予定の時間軸で管理する単一HTMLツール。
外部依存なし・オフライン動作。ブラウザで `index.html` を開くだけで使える。

## 主な機能

- **タイムライン表示**：細胞名でグループ化し、各培養を播種→回収予定のバーで表示。週末シェード・本日ライン付き。
- **本日のタスク**：今日が播種日 / 回収予定日の培養を「播種」「回収 / 処理」に分けてリスト表示（複数の用途もチップで表示）。
- **記録項目**：細胞名 / ディッシュ（100 mm・6・12・24・96 well）/ Passage / 播種日 / 回収予定日 / 遺伝子型（WT・KO＋遺伝子）/ 培地 / 播種濃度（数値＋10^n、または希釈 1:x）/ ボリューム（mL・/well）/ 添加薬品 / 用途（複数選択：継代・ISOGEN回収・DNA抽出・凍結ストック・その他 …追加可）/ メモ。
- **設定（マスタ管理）**：細胞名・培地・添加薬品・KO遺伝子・用途を追加/削除。
- **入出力**：Excel(CSV, UTF-8 BOM) と JSON で書き出し・読み込み。
- **保存**：入力データはブラウザの localStorage に自動保存。移行は CSV / JSON で行う。
- **表示**：ライト/ダーク対応、ズーム（1日あたりの表示幅）調整。

## 使い方

1. `index.html` をブラウザ（Chrome / Edge 等）で開く。
2. 「培養を追加」から播種を登録。バーやリストをクリックで編集。
3. 「設定」でマスタ（細胞名・培地・薬品・遺伝子・用途）を整備。
4. 「Excel書出 / 読込」でデータをバックアップ・移行。
